Transcription pointers:
  • there is no domain-specific spell-checking at this point.
  • non-lexical items should go in square brackets.
    • common ones: [NOISE], [CLICK], [SPEECH], [UH], [UM], [SIL], [SYNTH], etc.
    • there is no definitive set of non-lexicals; you're allowed to make them up... be conservative.
  • words that are cut off should be pre/post pended with a hyphen (-ISTEN, LISTE-, -ISTE-)
